Obstacle Recognition for Lower-Limb Exoskeleton Control using Computer Vision and Deep Learning

In this project, we developed a computer vision and AI-based perception system for obstacle detection in lower-limb exoskeletons. Using a combination of depth sensors, LiDAR, and machine learning (ML) algorithms, the research focuses on real-time object recognition to enhance exoskeleton-assisted mobility. A key component is sensor fusion, which integrates multiple sensor inputs to improve detection accuracy and adaptability in various environments. Additionally, Convolutional Neural Networks (CNNs) are used to process visual data, allowing the exoskeleton to recognize and classify obstacles, which contributes to safer and more adaptive robotic-assisted movement. Experimental and test results show that the system is accurate and reliable in terms of obstacle detection and distance measurement, demonstrating its potential for further development.

A Co-Simulation Approach of Multibody Dynamics of a Lower-Limb Exoskeleton

This research focuses on co-simulation techniques that integrate multiple simulation models to create a more accurate and efficient digital representation of complex physical systems. In order to have different simulation tools (e.g., finite element analysis, multibody dynamics, and control simulations) to exchange data in real time, we explored how co-simulation can improve accuracy, computational efficiency, and predictive capabilities. The research is particularly relevant for applications requiring dynamic, multi-domain interactions, where linking separate physics-based models enhances system performance and real-time decision-making.

This study demonstrates that co-simulation can improve the reliability of predictive models while reducing computational costs, making it a valuable approach for systems that require real-time responsiveness. However, challenges such as synchronization issues, data exchange latency, and computational resource demands, need to be addressed for more widespread adoption.

Enhancing Exoskeleton-Based Rehabilitation with Machine Learning and Sensor Optimization

Restoring mobility for patients with neurological impairments requires precise motion prediction and adaptive control. ML integration is therefore essential for effective rehabilitative exoskeletons. In this project, we explored ML and high-density EMG sensor optimization to enhance motion intent detection and real-time assistance, which improves human-robot interaction (HRI) in rehabilitation. We tackled two key challenges: accurate motion prediction for assistive control and optimized sensor placement for high-quality EMG signals, both crucial for exoskeleton adaptability and effectiveness.

For real-time gait prediction and motion assistance, we investigated K-Means clustering to classify gait phases and sequence-to-sequence neural networks for EMG signal mapping and domain adaptation. The model predicts optimal knee angles to assist movement while minimizing metabolic cost. An assist-as-needed (AAN) approach was implemented to provide necessary support while keeping patients engaged.

Since signal quality is crucial for accurate motion prediction, electrode placement in high-density EMG acquisition plays a significant role. Using Temporal Convolutional Networks (TCN), we evaluated optimal vs. standard electrode locations based on consistency metrics (signal area, peak height, location repeatability). Our findings indicate that electrode placement significantly affects model performance, and that the optimal location varies across conditions.
